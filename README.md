## Description

Project to parse json in email as attachment or get by url inside the email.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start
```

## Run tests

```bash
# unit tests
$ npm run test

```
